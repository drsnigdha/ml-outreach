Limits of Classical Logic
The Problem with Exploding Rule Trees

Traditional Coding Has Limits

Humans complete complex tasks naturally without thinking, but programming rigid logical rules for them is practically impossible.

Mapping faces dynamically across endless camera angles and lighting conditions.
Deciphering colloquial structures, sarcasm, and open cultural context.
Extracting speaker emotion from tiny vocal adjustments.
The Rule Explosion Problem

Writing unique nested if / else conditions for every potential facial coordinate shadow or vocal tone pitch would take infinite lines of code.

AI Solution: Instead of writing explicit logical rules, we let algorithms analyze patterns and learn the classification rules themselves.

Computing Models
Hard Computing Specifications

What Is Hard Computing?

Hard computing requires completely exact inputs and rules to produce precise, deterministic mathematical outputs.

Absolute Precision

Demands 100% exact answers; even tiny input errors will break execution.

Deterministic Behavior

Processing the same input through the code will always produce the identical output.

Zero Self-Learning

Systems follow static rules and cannot adapt to new conditions without manual code updates.

Standard Applications:

1. Digital Calculators

Multiplication and arithmetic operations require absolute mathematical accuracy.

2. ATM PIN Validation

Access is granted based on an exact matching string, with no tolerance for variations.

Slide 19: Classical Hard Computing
Slide 19 / 25
Computing Models
Soft Computing Specifications

Standard Applications:

1. Conversational Large Language Models

Chatbots generate paragraphs organically; there is no single "perfect" response.

2. Face Identification

Algorithms identify key structural landmarks despite changing light or angles.

What Is Soft Computing?

Soft computing tolerates approximate values and uncertainty, allowing systems to solve complex, unpredictable problems.

Approximate Answers

Focuses on generating high-probability solutions rather than 100% exact values.

Adaptive Feedback Learning

Models adjust their weights based on feedback loops to improve predictions over time.

Resilience to Noise

Successfully processes noisy, incomplete, or distorted real-world datasets.
